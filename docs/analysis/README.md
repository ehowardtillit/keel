# Analysis Documents

Analysis docs are created for **Tier S** changes -- investigations that need to happen before implementation.

## Template

Copy `000-template.md` and increment the number.

## When to create an analysis

- Database schema changes
- Security model changes
- Infrastructure / deployment changes
- Any change where multiple approaches exist and the trade-offs need evaluation
