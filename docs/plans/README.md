# Implementation Plans

Plans are created **before** implementation begins. They capture intent, approach, and scope so that:
- AI agents have context on what you're building
- Future developers understand *why* the code looks the way it does
- You think through the approach before writing code

## Template

Copy `000-template.md` and increment the number.

## When to create a plan

| Tier | Plan required? |
|------|---------------|
| **S** | Yes -- always |
| **A** | Yes -- always |
| **B** | Recommended |
| **C** | No |
