# Implementation Records

Records are created **after** implementation. They capture what was actually built, decisions made along the way, and lessons learned. These serve as rich context for AI agents and future developers.

## Template

Copy `000-template.md` and increment the number.

## When to create a record

| Tier | Record required? |
|------|-----------------|
| **S** | Yes |
| **A** | Yes |
| **B** | Yes |
| **C** | No |
